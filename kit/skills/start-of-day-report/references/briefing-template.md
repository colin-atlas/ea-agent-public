# Briefing Template

Use this exact structure for the SOD draft. This is a **daily ops lock-in**, not a recap. Omit empty sections only if the source is unavailable, and explicitly note the gap.

---

```
**Daily Ops Lock-In — {Day, Month Date, Year}**

**Today's Shape**
{Key meetings, prep-required meetings, deep-work windows, notable schedule collisions, or "Calendar unavailable"}

**Inbox / Attention Flags**
{Only items that matter today, or "No inbox flags" / "Inbox source unavailable"}

**Carryovers / Open Loops**
{Only unresolved items that still matter today, or "No meaningful carryovers"}

**Sacred Six Focus**
{Concise status of the most relevant Sacred Six items for today, or "No Sacred Six items in play today"}

**[EXECUTIVE_FIRST_NAME] Top 3**
1. {#id: task title} — {why it matters today}
2. {#id: task title} — {why}
3. {#id: task title} — {why}

**[EA_FIRST_NAME] Top 3**
1. {task / owner item} — {why it matters today}
2. {task / owner item} — {why}
3. {task / owner item} — {why}

**[AGENT_NAME] Top 3**
1. {#id: task title} — {why it matters today}
2. {#id: task title} — {why}
3. {#id: task title} — {why}

**Dependencies / Handoffs**
- {Who needs what from whom}
- {Decision / unblock / follow-up}

**Risks / Flags**
- {Urgent issue, blocker, missing link, or timing risk}
- {Or "No major flags"}

**Proposed lock-in:**
{One concise recommended operating plan for [EXECUTIVE_FIRST_NAME], [EA_FIRST_NAME], and [AGENT_NAME] today.}

Want any changes before we lock it in?
```
