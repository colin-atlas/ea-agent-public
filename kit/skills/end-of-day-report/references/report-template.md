# EOD Template

Use this exact structure for the EOD draft. This is a **daily closeout / end-of-day lock-in**, not just a report. Omit sections only when the source is unavailable and explicitly note the gap.

---

```
**Daily Closeout — {Day, Month Date, Year}**

**Day verdict:**
{One blunt sentence: on track / mixed / slipped / strong progress / blocked}

**[EXECUTIVE_FIRST_NAME] Closeout**
- {What completed}
- {What remains open}
- {What needs attention next}

**[EA_FIRST_NAME] Closeout**
- {What moved}
- {What remains open}
- {What needed tomorrow / from [EXECUTIVE_FIRST_NAME]}

**[AGENT_NAME] Closeout**
- {What completed}
- {What remains in flight}
- {What decisions / inputs are needed}

**Sacred Six / Strategic Progress**
{Only the relevant signal for today}

**Carryforwards**
- {What carries to tomorrow}
- {Who owns it}
- {What status it should hold}

**Inbox / Unresolved Flags**
- {Only what still matters after today}
- {Or "No major remaining flags"}

**Tomorrow's Provisional Plan**
**[EXECUTIVE_FIRST_NAME]:**
1. {Likely top priority}
2. {Likely top priority}
3. {Likely top priority}

**[EA_FIRST_NAME]:**
1. {Likely operating priority}
2. {Likely operating priority}
3. {Likely operating priority}

**[AGENT_NAME]:**
1. {Likely top priority}
2. {Likely top priority}
3. {Likely top priority}

**Learnings to Capture**
- {Decision / pattern / correction / people insight / project-state change}
- {Or "No meaningful promotions today"}

Anything to adjust before I lock the day?
```
