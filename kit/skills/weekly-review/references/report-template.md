# Weekly Review Template

Use this exact structure for the weekly draft. This is a **weekly closeout + next-week lock-in**, not just a summary. Omit sections only when the source is unavailable and explicitly note the gap.

---

```
**Weekly Review & Next-Week Lock-In — Week of {Mon Date} - {Fri Date}**

**Week verdict:**
{One blunt sentence: strong / mixed / drifted / blocked / productive but uneven}

**Operating Triangle Weekly Closeout**
**[EXECUTIVE_FIRST_NAME]**
- {What moved}
- {What stalled}
- {What still needs executive attention}

**[EA_FIRST_NAME]**
- {What drove}
- {What remains open}
- {Where ops leverage or bottlenecks showed up}

**[AGENT_NAME]**
- {What got built / improved}
- {What remains in flight}
- {What needs confirmation / support}

**Big Three Review**
1. **[Goal 1]** — [RAG] {what moved, what didn't, what's next}
2. **[Goal 2]** — [RAG] {what moved, what didn't, what's next}
3. **[Goal 3]** — [RAG] {what moved, what didn't, what's next}

**Open-Item Triage**
- {Stale / blocked / needs-review item} — {recommendation}
- {Archive / delegate / keep / reframe}

**Next Week Preview**
- {Heavy meeting days}
- {Prep needs}
- {Time-sensitive deadlines}
- {Risks entering the week}

**Sacred Six**
{Prefer 6 directly, or at most 8 candidates if choice is still needed}
1. {Task} — {why it matters}
2. {Task} — {why}
3. {Task} — {why}
4. {Task} — {why}
5. {Task} — {why}
6. {Task} — {why}

**Next-Week Focus**
**[EXECUTIVE_FIRST_NAME]**
1. {Top weekly focus}
2. {Top weekly focus}
3. {Top weekly focus}

**[EA_FIRST_NAME]**
1. {Top weekly focus}
2. {Top weekly focus}
3. {Top weekly focus}

**[AGENT_NAME]**
1. {Top weekly focus}
2. {Top weekly focus}
3. {Top weekly focus}

**Learnings to Capture**
- {Decision / pattern / correction / people insight / project-state change}
- {Or "No meaningful promotions this week"}

**Continuity Check**
- Friday daily closeout: {present / missing}
- Weekly review artifact: {pending until lock-in / confirmed}
- Operational gaps: {none / list}

Anything to adjust before we lock next week?
```
